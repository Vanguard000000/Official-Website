My Goals

What I'm working toward and why I'm using AI to help me get there.

Why I Use LLMs to Code

Bridging Ideas and Reality Faster

Aerospace engineering is about turning concepts into functioning systems — whether it's a pocket telescope for TSA or a robot for FRC. I'm learning AI-assisted coding so I can prototype, iterate, and deploy projects faster than traditional learn-first-build-later approaches. LLMs let me ask "how do I build this?" in real time and get working code I can learn from, modify, and improve.

Learning by Doing, With a Copilot

I learn best by building real things. Rather than spending months on tutorials before touching a project, I use AI to guide me through the code as I write it. The LLM explains patterns, suggests fixes, and reviews my work — like having a senior engineer pair-program with me. This mirrors how modern aerospace teams work with simulation, automation, and collaborative tools.

Building a Portfolio That Proves Competence

Deploying this website from scratch — styling it, adding interactivity, debugging layout issues — shows that I can ship real software. By documenting the process and making the site itself the portfolio, I demonstrate both technical skills and the ability to leverage AI effectively. That's a skill colleges and teams like FRC #324 look for: someone who can use every tool available to solve problems.

Preparing for Aerospace, One Commit at a Time

Rocket guidance, satellite telemetry, flight software — all aerospace systems rely on code. Learning to write clean, functional JavaScript and HTML now builds the foundation for Python, C++, and the embedded systems I'll encounter in college and beyond. AI accelerates that learning curve without skipping the fundamentals, because I still have to understand every line the LLM generates.

Coding Timeline

HTML structure, semantic elements, basic CSS. Learning to ask LLMs for explanations, not just code.

Week 2: Styling & Layout

Flexbox, Grid, responsive design. Using AI to debug layout issues and learn modern CSS patterns.

Week 3: JavaScript Basics

Variables, functions, DOM manipulation. Having AI explain concepts and review my practice code.

Week 4: Integration & Deploy

Building complete features, deploying to Netlify/Vercel. Documenting the AI collaboration workflow.
