# Somya's Portfolio - AI-Assisted Coding Journey

A personal portfolio website documenting my one-month journey learning web development with AI assistance.

## About This Project

This website serves as both a **learning artifact** and a **showcase** of skills acquired during an intensive one-month course on web development fundamentals. Every line of code was written with the help of LLM coding assistants, making this project a demonstration of:

- **HTML5** - Semantic structure, accessibility best practices
- **CSS3** - Modern layout (Flexbox/Grid), responsive design, animations
- **Vanilla JavaScript** - DOM manipulation, event handling, modern ES6+ syntax
- **AI-Assisted Development** - Effective prompting, code review, debugging workflows

## Website Sections

| Section | Description |
|---------|-------------|
| **Hero** | Introduction with name, tagline, and call-to-action |

## One-Month Class Goals

### Week 1: Foundations
- HTML structure and semantic elements
- Basic CSS styling
- Learning to ask LLMs for *explanations*, not just code

### Week 2: Styling & Layout
- Flexbox and CSS Grid
- Responsive design principles
- Using AI to debug layout issues

### Week 3: JavaScript Basics
- Variables, functions, DOM manipulation
- Event handling
- Having AI explain concepts and review practice code

### Week 4: Integration & Deploy
- Building complete interactive features
- Deploying to Netlify/Vercel
- Documenting the AI collaboration workflow

## Getting Started
